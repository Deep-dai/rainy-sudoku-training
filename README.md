# Rainy 数独训练 MVP

这是一个面向手机和平板的本地数独练习小工具，重点支持 6 宫格和 9 宫格的经典数独训练。

## 已完成的 MVP 功能

- 6 宫格：6x6，2x3 宫，每行、每列、每宫使用 1-6。
- 9 宫格：9x9，3x3 宫，每行、每列、每宫使用 1-9。
- 6 宫格提供超级简单、非常简单、简单、高手难度。
- 9 宫格提供超级简单、非常简单、高手难度、超级高手，不再显示简单难度。
- 9 宫格超级简单和非常简单分别控制为 18-25、25-30 个空格；高手难度为 38-44 个空格。
- 9 宫格超级高手控制为 44-47 个空格（34-37 个提示数），保证可以仅用显性单数和隐性单数排除完成，同时至少需要一次隐性单数判断；目标是让偶尔玩数独、掌握基础排除的成年人约 20 分钟完成。
- 本地随机出题，并校验唯一解。
- 练习模式：支持主动提示和草稿候选数。
- 练习过程中不进行即时判错，不显示冲突、死局或错误标红，避免形成对系统纠错的依赖。
- 点击“交作业”后才统一检查；错误或未填格会标红，修正后可再次提交，全部正确时显示完成用时。
- 比赛模式：隐藏主动提示，支持可配置倒计时，交卷后统一判定。
- 支持关闭实时计时显示；后台仍记录用时，完成时弹出总用时。
- 点选空格后弹出触控数字键盘，点空白处自动收起。
- 擦除、草稿模式。
- 适合手机和平板的响应式界面。
- 支持 PWA 离线缓存，安装到 iPad 主屏幕后可离线使用。
- 正确完成题目后可获得与难度匹配的贴纸奖励，共五个等级、20 张贴纸。
- 9 宫格非常简单完成后有 80% 概率获得 4 级奖励、20% 概率获得 5 级奖励。
- 贴纸支持本地收藏、重复升级至三星闪卡，以及无提示完成的“独立完成”皇冠。
- 收藏页中已获得的贴纸可以点击查看大图。
- 二级、三级使用儿童绘本动物插画，四级使用珍藏毛绒主题和增强光效，五级超级珍藏使用小黑龙、小白虎、小黄鸭和光煞的专属全彩插画。

奖励规则和后续实施记录见 [`docs/reward-system-v1-plan.md`](docs/reward-system-v1-plan.md)。

## 使用方式

直接用浏览器打开 `index.html` 即可使用。

如果需要本地预览地址，可以在本目录启动静态服务，然后打开：

```bash
http://127.0.0.1:4173/index.html
```

## 发布到 GitHub Pages

项目已包含 `.github/workflows/pages.yml`。推送到 GitHub 的 `main` 分支后，GitHub Actions 会自动发布静态站点。

本仓库发布完成后的地址是：

```text
https://deep-dai.github.io/rainy-sudoku-training/
```

第一次发布通常需要等 1-3 分钟。进入仓库的 `Actions` 页面可以查看部署进度。

## 在 iPad 上使用

### 临时方式：同 Wi-Fi 访问 Mac

让 Mac 和 iPad 连接同一个 Wi-Fi，然后在 Mac 上启动本地服务：

```bash
python3 -m http.server 4173 --bind 0.0.0.0
```

再查看 Mac 的局域网 IP，例如 `192.168.1.23`。在 iPad 的 Safari 里打开：

```text
http://192.168.1.23:4173/index.html
```

如果想长期使用，可以把这几个静态文件部署到 GitHub Pages、Netlify、Vercel 或家里的 NAS。部署后 iPad 直接打开网址即可。

### 推荐方式：安装成 iPad 离线 App

把这些文件部署到支持 HTTPS 的静态网站，例如 GitHub Pages、Netlify 或 Vercel。然后在 iPad Safari 里打开部署后的网址，点分享按钮，选择“添加到主屏幕”。

第一次打开时需要联网。安装完成并成功打开一次后，页面、样式和脚本会缓存到 iPad，本工具可以离线运行。

注意：iPad Safari 的离线缓存需要 HTTPS。通过 `http://192.168.x.x:4173` 这种局域网地址访问时，可以练习，但通常不能完成可靠的离线安装。

### 真正完全本地的方式

如果不想依赖任何网址，可以把这套网页打包成一个 iPad 原生壳应用，例如用 Capacitor 或 Xcode 的 WKWebView。这样可以像普通 iPad App 一样安装和运行，但需要 Apple 开发者工具链。
